普通细胞规则

邻居数 < 2 → 死亡（人口过少）
邻居数 = 2 或 3 → 存活
邻居数 > 3 → 死亡（人口过多）
死细胞邻居数 = 3 → 新生为普通细胞

细胞S特殊规则

计算邻居时视为两个活细胞
邻居权重在2-4之间 → 存活
死细胞复活时，如果3个活细胞贡献中至少有两个是细胞S → 新生为细胞S
普通细胞与细胞S相邻 → 20%概率转变为细胞S
